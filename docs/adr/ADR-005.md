# Architecture Decision Record: ADR-005 — Why Claude as primary LLM Core (Served via Enterprise Gemini Proxy)

## Status
Approved

## Context
CareerOS AI requires a powerful, highly creative, and instruction-compliant Reasoning Large Language Model to compose customized matching packages, draft tailored emails, and sequence multi-stage educational plans.

## Decision
Claude (specifically Claude 3.5 Sonnet) served via our highly reliable proxy is selected as our core model due to the following characteristics:

1. **Strict Output-Format Compliance**: Claude displays unmatched reliability in parsing complex instruction sets and returning valid, structured JSON schemas without escaping out of specified parameters.
2. **Exemplary Composition Nuance**: Emits mature, personalized professional prose that avoids robotic, repetitive corporate language ("pleased to introduce", "delve", "testament to").
3. **Massive Context Windows**: 200,000 token boundaries allow candidates to upload massive, multi-page knowledge document archives for single-prompt semantic synthesis.
4. **Agentic Autonomy Performance**: Excels in multi-step reasoning, tool invocations, and API-parameter construction loops within our LangGraph state channels.

## Consequences
- **Positive**: Fluent composition quality, precise structured outputs, and strong reasoning capabilities.
- **Negative**: High token pricing compared to lightweight alternatives and strict ingestion rate-limit rules.
