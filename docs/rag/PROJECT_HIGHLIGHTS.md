# Project Highlights

Last verified from source code: 2026-06-14

## Why the project is impressive

CareerOS is not just a CRUD app.
It combines a real product surface with AI reasoning, retrieval, orchestration, voice delivery, docs RAG, and governance.

## AI engineering skills it proves

- Retrieval-backed application design
- Docs RAG indexing and citation-backed answering
- Prompt registry and prompt versioning
- Multi-agent decomposition
- Confidence and governance gating
- LLM fallback design with `FallbackProvider`
- Voice synthesis and outbound call orchestration
- Memory and outcome intelligence

## Backend skills it proves

- FastAPI router design
- Async Python and SQLAlchemy
- Repository and service layering
- Pydantic request and response modeling
- Background job orchestration
- Health and dependency checks

## Frontend skills it proves

- Next.js App Router structure
- Role-aware navigation and shell design
- Realtime websocket UX
- Auth session handling
- Mentor/HR chatbot page with citations and confidence display
- Data-fetching hook composition

## Automation and orchestration skills it proves

- LangGraph workflow design
- Autonomous action gating
- Human approval queues
- MCP-governed external tool usage
- Make.com-compatible webhook orchestration for docs RAG
- Traceability across session, event, and log records
